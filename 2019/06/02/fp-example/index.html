一个函数式的flickr

这个应用将做4件事:

  1. 根据特定搜索关键字构造url
  2. 向flickr发送api请求
  3. 把返回的json转为html
  4. 把图片展示出来

以上动作中有两个是不纯的, 包括从flickr的api获取数据和把图片展示到屏幕上. 我们就先定义这两个动作, 这样就能隔离它们了.
